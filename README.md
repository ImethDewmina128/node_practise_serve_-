# NODE_PRACTTISE_SERVE

This project consists of a static frontend and a mock backend server.

## Project Structure

- **mock-srv/**: Contains the Node.js mock server application.
  - **plugins/**: Server plugins (e.g., data utilities, support).
  - **routes/**: API route definitions (e.g., confectionery, electronics).
  - **test/**: Unit and integration tests.
- **static/**: Contains the frontend static files.
  - `index.html`: Main entry point.
  - `app.js`: Frontend logic.

## Getting Started

### Prerequisites

- Node.js
- npm

### Running the Mock Server

1. Navigate to the server directory:
   ```bash
   cd mock-srv
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   npm start
   ```

### Running the Frontend

Open `static/index.html` in your browser or serve the `static` directory using a simple HTTP server.

## API Endpoints

The mock server provides the following endpoints:

- `/` (Root)
- `/confectionery`
- `/electronics`
- `/example`// filepath: README.md
